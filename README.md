# Vendesk (CRM).
El paquete Vendesk (CRM) de V12 es el módulo especializado en la gestión de la relación con clientes, ofreciendo herramientas para administrar atenciones, casos (tickets), prospectos y oportunidades de venta. Su propósito es fortalecer la interacción con clientes y prospectos, mejorar la atención y facilitar el seguimiento de oportunidades comerciales dentro del ERP.
